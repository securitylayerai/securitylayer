Security Layer ist eine Sicherheitsplattform für Agenten, die fähigkeitsbasierte Zugriffskontrolle, Taint-Tracking, Aktionsnormalisierung und deterministische Regeln durchsetzt — damit gefährliche Aktionen **strukturell unmöglich** sind, nicht nur erkannt werden.
