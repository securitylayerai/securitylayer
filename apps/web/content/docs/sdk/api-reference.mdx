## SecurityLayer

The main SDK entry point.

```typescript
import { SecurityLayer } from "@securitylayerai/sdk";

const sl = new SecurityLayer(options?: SecurityLayerOptions);
```

### Options

| Option | Type | Default | Description |
|---|---|---|---|
| `configDir` | `string` | `~/.securitylayer` | Path to Security Layer configuration directory |
| `caller` | `string` | Auto-detected | Identifier for the calling application |
| `sessionId` | `string` | Auto-generated | Default session ID for checks |

## Methods

### `check()`

Evaluate an action through all 3 security layers before execution.

```typescript
const result = await sl.check(
  tool: string,
  params: Record<string, unknown>,
  context?: ExecutionContext,
): Promise<CheckResult>;
```

**Parameters:**

| Parameter | Type | Description |
|---|---|---|
| `tool` | `string` | The capability being used: `"exec"`, `"file.read"`, `"file.write"`, `"web_fetch"`, etc. |
| `params` | `object` | Tool-specific parameters (e.g., `{ command: "ls -la" }` for exec) |
| `context` | `ExecutionContext` | Optional session context for the check |

**Returns:** `Promise<CheckResult>`

### `waitForApproval()`

Block until a human approves or denies a `REQUIRE_APPROVAL` decision.

```typescript
const approved = await sl.waitForApproval(
  approvalId: string,
  options?: ApprovalOptions,
): Promise<boolean>;
```

**Parameters:**

| Parameter | Type | Description |
|---|---|---|
| `approvalId` | `string` | The approval ID from a `REQUIRE_APPROVAL` result |
| `options.timeout` | `number` | Timeout in milliseconds (default: 300000 / 5 minutes) |

**Returns:** `true` if approved, `false` if denied or timed out.

### `ingestContent()`

Update the taint tracker when external content enters the session.

```typescript
sl.ingestContent(
  content: string,
  metadata: ContentMetadata,
): void;
```

**Parameters:**

| Parameter | Type | Description |
|---|---|---|
| `content` | `string` | The content that was read/fetched |
| `metadata.source` | `"web" \| "file"` | Where the content came from |
| `metadata.path` | `string?` | File path (for file sources) |
| `metadata.url` | `string?` | URL (for web sources) |

Calling this elevates the session taint level based on the source. Web content elevates to `WEB_CONTENT`, untrusted project files elevate to `UNTRUSTED_HUMAN`.

### `scanEgress()`

Scan outbound content for leaked secrets, PII, and sensitive data before sending.

```typescript
const result = sl.scanEgress(content: string): EgressScanResult;
```

**Returns:** `EgressScanResult` with `clean: boolean` and `findings[]`.

### `getSessionTaint()`

Get the current session's taint level.

```typescript
const taint = sl.getSessionTaint(): TaintLevel;
```

**Returns:** One of `"owner"`, `"trusted"`, `"untrusted"`, `"web"`, `"skill"`, `"memory"`.

## Types

### CheckResult

```typescript
interface CheckResult {
  decision: "ALLOW" | "DENY" | "REQUIRE_APPROVAL";
  reason?: string;
  approvalId?: string;
  layers: {
    capability: { allowed: boolean; reason?: string };
    taint: { level: TaintLevel; allowed: boolean; reason?: string };
    rules: { decision: string; reason?: string } | null;
    llm: {
      decision: string;
      confidence: number;
      reasoning?: string;
    } | null;
  };
  degraded: boolean;
  timing: {
    capability_ms: number;
    taint_ms: number;
    normalize_ms: number;
    rules_ms: number;
    llm_ms: number | null;
    total_ms: number;
  };
}
```

### EgressScanResult

```typescript
interface EgressScanResult {
  clean: boolean;
  findings: EgressFinding[];
}

interface EgressFinding {
  type:
    | "api_key"
    | "pem_block"
    | "private_key"
    | "password"
    | "high_entropy"
    | "pii"
    | "credential_pattern";
  confidence: number;
  location: { start: number; end: number };
  redacted: string;
}
```

### TaintLevel

```typescript
type TaintLevel =
  | "owner"     // Direct from authenticated owner
  | "trusted"   // From allowlisted contact
  | "untrusted" // From unknown/unverified human
  | "web"       // From web fetch, browser, email
  | "skill"     // Generated by a skill
  | "memory";   // Loaded from persistent memory
```

### ExecutionContext

```typescript
interface ExecutionContext {
  sessionId: string;
  taint?: TaintLevel;
  caller?: string;
  skillId?: string;
  channelId?: string;
}
```
