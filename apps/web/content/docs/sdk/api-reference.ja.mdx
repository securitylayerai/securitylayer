## createSecurityLayer()

`SecurityLayer` インスタンスを作成して返します。これが SDK のメインエントリーポイントです。

```typescript
import { createSecurityLayer } from "@securitylayerai/sdk";

const sl = await createSecurityLayer(options?: SecurityLayerOptions);
```

### オプション

| オプション | 型 | デフォルト | 説明 |
|---|---|---|---|
| `configDir` | `string` | `~/.securitylayer` | Security Layer 設定ディレクトリのパス |
| `sessionId` | `string` | 自動生成 UUID | チェック用のデフォルトセッション ID |
| `config` | `LoadedConfig` | — | プリロードされた設定オブジェクト（ファイル読み込みをスキップ、テストに便利） |
| `eventBus` | `EventBus` | 自動作成 | 注入されたイベントバス（テストやイベント共有に便利） |

**スロー:** 設定ファイルをロードできない場合は `ConfigError`、security pipeline の初期化に失敗した場合は `InitializationError`。

## SecurityLayer

`createSecurityLayer()` が返すオブジェクトです。セキュリティチェック、taint tracking、イベントハンドリングの全メソッドを提供します。

### check()

実行前にアクションを security pipeline で評価します。

```typescript
const result = await sl.check(
  tool: string,
  params: Record<string, unknown>,
  context?: ExecutionContext,
): Promise<CheckResult>;
```

**パラメータ:**

| パラメータ | 型 | 説明 |
|---|---|---|
| `tool` | `string` | 使用する capability: `"exec"`, `"file.read"`, `"file.write"`, `"web_fetch"` など |
| `params` | `object` | ツール固有のパラメータ。exec の場合、コマンドは `command`、`cmd`、または `input` プロパティから抽出されます。 |
| `context` | `ExecutionContext` | このチェックに対するオプショナルなコンテキストオーバーライド |

**戻り値:** `Promise<CheckResult>`

### waitForApproval()

`REQUIRE_APPROVAL` 判定に対して、人間が承認または拒否するまでブロックします。

```typescript
const approved = await sl.waitForApproval(
  approvalId: string,
  options?: ApprovalOptions,
): Promise<boolean>;
```

**パラメータ:**

| パラメータ | 型 | 説明 |
|---|---|---|
| `approvalId` | `string` | `REQUIRE_APPROVAL` チェック結果からの承認 ID |
| `options.timeout` | `number` | タイムアウト（ミリ秒、デフォルト: 300000 / 5 分） |

**戻り値:** 承認された場合 `true`、拒否・タイムアウト・`approvalId` が不明な場合は `false`。

### ingestContent()

外部コンテンツがセッションに入った際に taint トラッカーを更新します。

```typescript
sl.ingestContent(
  content: string,
  metadata: ContentMetadata,
): void;
```

**パラメータ:**

| パラメータ | 型 | 説明 |
|---|---|---|
| `content` | `string` | 読み取りまたはフェッチされたコンテンツ |
| `metadata.source` | `"web" \| "file" \| "channel" \| "skill" \| "memory"` | コンテンツの取得元 |
| `metadata.path` | `string?` | ファイルパス、スキル名、またはチャネル識別子 |
| `metadata.url` | `string?` | URL（Web ソースの場合） |

**ソースから taint level へのマッピング:**

| ソース | Taint Level |
|---|---|
| `web` | WEB |
| `file` | UNTRUSTED |
| `channel` | UNTRUSTED |
| `skill` | SKILL |
| `memory` | MEMORY |

### scanEgress()

送信前に、送信コンテンツに漏洩した秘密情報、PII、機密データがないかスキャンします。

```typescript
const result = sl.scanEgress(content: string): EgressScanResult;
```

**戻り値:** `clean: boolean` と `findings: EgressFinding[]` を含む `EgressScanResult`。

### getSessionTaint()

現在のセッションの有効な taint level を取得します。

```typescript
const taint = sl.getSessionTaint(): string;
```

**戻り値:** `"owner"`, `"trusted"`, `"untrusted"`, `"web"`, `"skill"`, `"memory"` のいずれか。

### on()

内部イベントバスからセキュリティイベントをサブスクライブします。

```typescript
const unsubscribe = sl.on(type: string, handler: (event) => void): () => void;
```

**パラメータ:**

| パラメータ | 型 | 説明 |
|---|---|---|
| `type` | `string` | リッスンするイベントタイプ（例: `"action.evaluated"`, `"taint.elevated"`, `"approval.requested"`） |
| `handler` | `function` | イベント発生時に呼び出されるコールバック |

**戻り値:** アンサブスクライブ関数。呼び出すとイベントの受信を停止します。

### destroy()

リソースをクリーンアップします。保留中のすべての承認リクエストをタイムアウトとして解決し、すべてのイベントハンドラーをクリアします。

```typescript
sl.destroy(): void;
```

SecurityLayer インスタンスの使用が完了したら、必ず `destroy()` を呼び出してください。

## withSecurityLayer()

任意の関数にセキュリティチェックを追加するミドルウェアラッパーです。呼び出しを透過的にインターセプトします — ラップされた関数は元の関数と同じシグネチャを持ちます。

```typescript
import { createSecurityLayer, withSecurityLayer } from "@securitylayerai/sdk";

const sl = await createSecurityLayer();

const protectedExec = withSecurityLayer(sl, originalExecFn, "exec");
const protectedRead = withSecurityLayer(sl, originalReadFn, "file.read");
```

**シグネチャ:**

```typescript
function withSecurityLayer<TArgs extends unknown[], TReturn>(
  sl: SecurityLayer,
  fn: (...args: TArgs) => TReturn | Promise<TReturn>,
  tool: string,
  options?: {
    extractParams?: (...args: unknown[]) => Record<string, unknown>;
  },
): (...args: TArgs) => Promise<TReturn>;
```

**動作:**
- `ALLOW` → 元の関数を呼び出し、その結果を返します
- `DENY` → 関数を呼び出さずに `SecurityLayerError` をスローします
- `REQUIRE_APPROVAL` → 承認を待ち、拒否された場合はスローします

**デフォルトのパラメータ抽出:** 最初の引数が文字列の場合、`{ command: arg }` にマッピングされます。オブジェクトの場合、そのまま params として使用されます。

## エラークラス

すべての SDK エラーは `SecurityLayerError` を継承し、`SecurityLayerError` は `Error` を継承しています。

```typescript
import {
  SecurityLayerError,
  ConfigError,
  InitializationError,
  CheckError,
  ApprovalTimeoutError,
} from "@securitylayerai/sdk";
```

| エラー | プロパティ | スローされる条件 |
|---|---|---|
| `SecurityLayerError` | `message` | 基底クラス。`withSecurityLayer()` による DENY 時にもスローされます |
| `ConfigError` | `configPath?: string` | 設定ファイルをロードまたはパースできない場合 |
| `InitializationError` | `cause?: unknown` | Security pipeline の初期化に失敗した場合 |
| `CheckError` | `tool?: string` | アクション評価が予期せず失敗した場合 |
| `ApprovalTimeoutError` | `approvalId: string` | 承認リクエストがタイムアウトした場合 |

## 型

### CheckResult

```typescript
interface CheckResult {
  decision: "ALLOW" | "DENY" | "REQUIRE_APPROVAL";
  reason: string;
  approvalId?: string;
  layers: LayerResults;
  degraded: boolean;
  timing: {
    total: number;
    capability?: number;
    rules?: number;
    llm?: number;
  };
}
```

### LayerResults

```typescript
interface LayerResults {
  capability: {
    allowed: boolean;
    reason?: string;
  };
  rules?: {
    decision: string;
    reason?: string;
  };
  llm?: {
    decision: string;
    confidence: number;
    reasoning?: string;
  };
}
```

### EgressScanResult

```typescript
interface EgressScanResult {
  clean: boolean;
  findings: EgressFinding[];
}

interface EgressFinding {
  type:
    | "api_key"
    | "pem_block"
    | "private_key"
    | "password"
    | "high_entropy"
    | "pii"
    | "credential_pattern";
  confidence: number;
  location: { start: number; end: number };
  redacted: string;
}
```

### ExecutionContext

```typescript
interface ExecutionContext {
  sessionId?: string;
  skillId?: string;
  channelId?: string;
}
```

### ContentMetadata

```typescript
interface ContentMetadata {
  source: "web" | "file" | "channel" | "skill" | "memory";
  path?: string;
  url?: string;
}
```

### SecurityLayerOptions

```typescript
interface SecurityLayerOptions {
  configDir?: string;
  sessionId?: string;
  config?: LoadedConfig;
  eventBus?: EventBus;
}
```

### ApprovalOptions

```typescript
interface ApprovalOptions {
  timeout?: number; // milliseconds, default 300000 (5 min)
}
```
