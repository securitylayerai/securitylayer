## 사전 요구 사항

- Security Layer CLI가 설치되고 초기화되어 있어야 합니다 (`sl init`)
- Node.js 18+ 또는 Bun 1.0+

## 기본 설정

```typescript
import { SecurityLayer } from "@securitylayerai/sdk";

// 기본 설정으로 초기화합니다 (~/.securitylayer/ 를 읽습니다)
const sl = new SecurityLayer();

// 또는 명시적 설정으로 초기화합니다
const sl = new SecurityLayer({
  configDir: "/path/to/.securitylayer",
  caller: "my-agent",
  sessionId: "session-123",
});
```

## 첫 번째 검사

에이전트가 수행하는 모든 도구 호출은 실행 전에 `sl.check()`를 통과해야 합니다:

```typescript
async function runTool(tool: string, params: Record<string, unknown>) {
  const result = await sl.check(tool, params, {
    sessionId: "current-session",
    taint: sl.getSessionTaint(),
    caller: "my-agent",
  });

  switch (result.decision) {
    case "ALLOW":
      // 실행해도 안전합니다
      return executeToolCall(tool, params);

    case "DENY":
      // 구조적으로 차단됨 — 실행하지 마세요
      throw new Error(`Security Layer: ${result.reason}`);

    case "REQUIRE_APPROVAL":
      // 진행하기 전에 사람의 승인이 필요합니다
      const approved = await sl.waitForApproval(result.approvalId);
      if (!approved) throw new Error("승인이 거부되었거나 시간이 초과되었습니다");
      return executeToolCall(tool, params);
  }
}
```

## 결정 이해하기

모든 검사는 3개 security layers의 결정이 포함된 `CheckResult`를 반환합니다:

```typescript
const result = await sl.check("exec", { command: "git push --force" });

// 어떤 레이어가 차단했는지 확인합니다
console.log(result.layers.capability); // { allowed: true }
console.log(result.layers.taint);      // { level: "owner", allowed: true }
console.log(result.layers.rules);      // { decision: "DENY", reason: "Force push" }
console.log(result.layers.llm);        // { decision: "NORMAL", confidence: 0.9 }

// 성능
console.log(result.timing.total_ms);   // 12
console.log(result.degraded);          // false (LLM을 사용할 수 있었음)
```

## Taint 추적

에이전트가 외부 콘텐츠를 읽을 때 taint 추적기를 업데이트합니다:

```typescript
// 웹 페이지를 읽은 후
const webContent = await fetch("https://example.com").then(r => r.text());
sl.ingestContent(webContent, {
  source: "web",
  url: "https://example.com",
});

// 신뢰할 수 없는 프로젝트의 파일을 읽은 후
const fileContent = await readFile("/tmp/downloaded-project/config.js", "utf-8");
sl.ingestContent(fileContent, {
  source: "file",
  path: "/tmp/downloaded-project/config.js",
});

// 이후 검사에 상승된 taint가 반영됩니다
const taint = sl.getSessionTaint(); // "web" ("owner"에서 상승됨)
```

## 발신 콘텐츠 스캔

에이전트가 데이터를 외부로 전송하기 전에 유출된 비밀을 스캔합니다:

```typescript
const message = agent.generateResponse();

const scan = sl.scanEgress(message);
if (!scan.clean) {
  console.error("비밀 감지:", scan.findings.map(f => f.type));
  // 메시지를 전송하지 마세요 — 민감한 데이터가 포함되어 있습니다
}
```
