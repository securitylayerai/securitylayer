Generates shell completion scripts for tab-completion of Security Layer commands and arguments.

## Usage

```bash
securitylayer completions <shell>
```

## Arguments

| Argument | Default | Description |
|---|---|---|
| `<shell>` | `bash` | Shell type: `bash`, `zsh`, or `fish` |

## Installation

### Bash

```bash
securitylayer completions bash > /etc/bash_completion.d/securitylayer
```

### Zsh

```bash
securitylayer completions zsh > "${fpath[1]}/_securitylayer"
```

### Fish

```bash
securitylayer completions fish > ~/.config/fish/completions/securitylayer.fish
```

## Exit codes

| Code | Meaning |
|---|---|
| `0` | Completion script output successfully |
| `1` | Unsupported shell |
