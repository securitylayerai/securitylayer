学習済みルール（「承認して記憶」で自動承認されたパターン）を管理します。

## 使い方

```bash
securitylayer rules list
securitylayer rules revoke <number>
securitylayer rules clear
```

## サブコマンド

### `rules list`

`~/.securitylayer/learned-rules.json` のすべての学習済みルールを表示します。

```bash
$ securitylayer rules list

  Learned rules:
    1. npm install * (exec) — 2025-01-15 — claude-code
    2. git push * (exec) — 2025-01-16 — claude-code
```

各エントリにはパターン、ケイパビリティ、作成日、セッション ID が表示されます。

### `rules revoke <number>`

番号（`rules list` の 1 始まりのインデックス）を指定して、特定の学習済みルールを削除します。

```bash
$ securitylayer rules revoke 2
Revoked rule: git push * (exec)
```

**終了コード:** 成功時 `0`、無効なルール番号または引数不足の場合 `1`。

### `rules clear`

すべての学習済みルールを削除します。

```bash
$ securitylayer rules clear
Cleared 2 learned rule(s).
```

**終了コード:** 常に `0`。
