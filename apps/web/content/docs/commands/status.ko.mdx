Security Layer 설정, 통합 상태 및 활성 세션의 개요를 표시합니다.

## 사용법

```bash
securitylayer status
```

## 출력

상태 대시보드에 표시되는 항목:

- **설정** — 설정 파일의 로드 여부 및 위치
- **Shield 상태** — 활성 또는 비활성
- **Claude Code 훅** — 설치됨 또는 미설치
- **세션** — 설정된 세션 수 및 세부 정보
- **학습 모드** — 활성 기간 및 만료 시간 (활성화된 경우)

## 출력 예시

```
Security Layer Status

  ✓ Configuration loaded
    config.yaml
    capabilities/sessions.yaml
    capabilities/channels.yaml
    ai-tools.yaml
    projects.yaml

  Shield: active
  Claude Code hooks: installed
  Sessions: 1
    claude-code (exec, file.read, file.write, web_fetch)
```

## 종료 코드

| 코드 | 의미 |
|---|---|
| `0` | 항상 성공적으로 종료됨 |
