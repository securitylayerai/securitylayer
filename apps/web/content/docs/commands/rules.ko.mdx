학습된 규칙 — "승인 후 기억하기"를 통해 자동 승인된 패턴을 관리합니다.

## 사용법

```bash
securitylayer rules list
securitylayer rules revoke <number>
securitylayer rules clear
```

## 하위 명령

### `rules list`

`~/.securitylayer/learned-rules.json`에 저장된 모든 학습된 규칙을 표시합니다.

```bash
$ securitylayer rules list

  Learned rules:
    1. npm install * (exec) — 2025-01-15 — claude-code
    2. git push * (exec) — 2025-01-16 — claude-code
```

각 항목에는 패턴, capability, 생성일, 세션 ID가 표시됩니다.

### `rules revoke <number>`

번호(`rules list`에서의 1-인덱스)로 특정 학습된 규칙을 제거합니다.

```bash
$ securitylayer rules revoke 2
Revoked rule: git push * (exec)
```

**종료 코드:** 성공 시 `0`, 잘못된 규칙 번호 또는 인자 누락 시 `1`.

### `rules clear`

모든 학습된 규칙을 제거합니다.

```bash
$ securitylayer rules clear
Cleared 2 learned rule(s).
```

**종료 코드:** 항상 `0`.
