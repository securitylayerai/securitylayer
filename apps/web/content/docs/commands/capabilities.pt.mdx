Mostra todas as capabilities concedidas organizadas por sessoes, canais e skills.

## Uso

```bash
sl capabilities show
```

## Saida

Exibe as capabilities agrupadas por:

- **Sessoes** — Lista de capabilities de cada sessao e taint level padrao
- **Canais** — Restricoes de capabilities por canal
- **Skills** — Declaracoes de capabilities por skill

## Exemplo

```bash
$ sl capabilities show

  Sessions:
    claude-code
      capabilities: exec, file.read, file.write, web_fetch
      default_taint: owner

  Channels:
    (none configured)

  Skills:
    (none configured)
```

## Codigos de saida

| Codigo | Significado |
|---|---|
| `0` | Sempre sai com sucesso |
