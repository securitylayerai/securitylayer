Simuliert die Richtlinienauswertung für einen Befehl, ohne ihn auszuführen. Zeigt die vollständigen Auswertungsergebnisse aller Schichten.

## Verwendung

```bash
sl policy check "<cmd>" [--session <name>]
```

## Flags

| Flag | Standard | Beschreibung |
|---|---|---|
| `--command <cmd>` | (positional) | Der zu simulierende Befehl |
| `--session <name>` | `claude-code` | Session, gegen die ausgewertet wird |

## Ausgabe

Der Befehl zeigt:

- **Eingabe** — Befehl, Session, Taint-Level, aktuelles Arbeitsverzeichnis
- **Schichtergebnisse** — Capability-Gate, Regel-Engine, LLM-Judge, Risikobewertung
- **Timing** — Aufschlüsselung der Zeitdauer pro Schicht
- **Endentscheidung** — `ALLOW`, `DENY` oder `REQUIRE_APPROVAL`

## Beispiel

```bash
$ sl policy check "rm -rf /"

  Command:  rm -rf /
  Session:  claude-code
  Taint:    OWNER
  CWD:      /home/user/project

  Layer Results:
    Capability gate: ALLOW (exec granted)
    Rules:           DENY (recursive delete of root)
    LLM judge:       DANGEROUS (confidence: 0.95)
    Risk score:      1.0

  Timing:
    Total:      45ms
    Capability: 0ms
    Rules:      2ms
    LLM:        43ms

  Decision: DENY
```

## Exit-Codes

| Code | Bedeutung |
|---|---|
| `0` | Beendet immer erfolgreich |
