显示按会话、通道和技能分组的所有已授权能力。

## 用法

```bash
sl capabilities show
```

## 输出

按以下分组显示能力：

- **会话** — 每个会话的能力列表和默认污点级别
- **通道** — 通道能力限制
- **技能** — 技能能力声明

## 示例

```bash
$ sl capabilities show

  Sessions:
    claude-code
      capabilities: exec, file.read, file.write, web_fetch
      default_taint: owner

  Channels:
    (none configured)

  Skills:
    (none configured)
```

## 退出码

| 代码 | 含义 |
|---|---|
| `0` | 始终成功退出 |
