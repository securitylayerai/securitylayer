AI 도구와의 Security Layer 통합을 자동으로 설정합니다.

## 사용법

```bash
sl setup claude-code
sl setup cursor
```

## 하위 명령

### `setup claude-code`

`~/.claude/hooks.json`에 Security Layer 훅을 설치합니다.

**PreToolUse 훅** (도구 실행 전 평가):
- `Bash` — 셸 명령 실행
- `Write` — 파일 생성
- `Edit` — 파일 수정
- `WebFetch` — URL 가져오기
- `NotebookEdit` — Jupyter 노트북 편집

**PostToolUse 훅** (도구 완료 후 실행):
- `Bash` — 명령 실행 컨텍스트 추적
- `Read` — taint 상승을 위한 파일 읽기 추적
- `WebFetch` — taint 상승을 위한 웹 콘텐츠 가져오기 추적

각 훅은 적절한 플래그와 함께 `sl hook claude-code`를 호출합니다.

**종료 코드:** 성공 또는 Claude Code를 찾지 못한 경우 `0`.

### `setup cursor`

Cursor 보호를 위한 안내를 제공합니다. 네이티브 훅 통합은 v1에서 계획되어 있으며, 현재는 셸 shim을 사용합니다:

```bash
sl shield enable
```

**종료 코드:** 항상 `0`.
