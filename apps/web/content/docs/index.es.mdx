Security Layer es una plataforma de seguridad para agentes que aplica control de acceso basado en capacidades, seguimiento de contaminación (taint), normalización de acciones y reglas determinísticas, para que las acciones peligrosas sean **estructuralmente imposibles**, no solo detectadas.
