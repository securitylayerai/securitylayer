Security Layer는 capability 기반 접근 제어, taint 추적, 동작 정규화, 결정적 규칙을 통해 위험한 동작을 단순히 탐지하는 것이 아니라 **구조적으로 불가능하게** 만드는 에이전트 보안 플랫폼입니다.
