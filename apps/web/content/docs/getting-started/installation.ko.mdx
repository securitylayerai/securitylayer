## 시스템 요구사항

- **런타임:** [Bun](https://bun.sh) v1.0 이상
- **OS:** macOS 또는 Linux
- **선택사항:** Docker (웹 대시보드 데이터베이스용)

## npm으로 설치

```bash
bun add -g securitylayer
```

또는 npm으로:

```bash
npm install -g securitylayer
```

## 설치 확인

```bash
securitylayer --version
```

짧은 별칭도 사용할 수 있습니다:

```bash
sl --version
```

## 포함 항목

`securitylayer` 패키지에 포함되는 항목:

- **`securitylayer` / `sl`** — CLI 바이너리
- **`@securitylayer/core`** — 보안 엔진 (capability 게이트, taint 추적기, 정규화기, 규칙, LLM 심판)

## 다음 단계

대화형 설정 마법사를 실행합니다:

```bash
securitylayer init
```

전체 안내는 [Init](/ko/docs/getting-started/init)을, 5분 만에 완료하는 가이드는 [빠른 시작](/ko/docs/getting-started/quick-start)을 참고하세요.
