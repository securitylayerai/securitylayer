## System requirements

- **Runtime:** [Bun](https://bun.sh) v1.0 or later
- **OS:** macOS or Linux
- **Optional:** Docker (for the web dashboard database)

## Install the CLI

<Tabs items={["npm", "yarn", "pnpm", "bun"]}>
  <Tab value="npm">
    ```bash
    npm install -g securitylayer
    ```
  </Tab>
  <Tab value="yarn">
    ```bash
    yarn global add securitylayer
    ```
  </Tab>
  <Tab value="pnpm">
    ```bash
    pnpm add -g securitylayer
    ```
  </Tab>
  <Tab value="bun">
    ```bash
    bun add -g securitylayer
    ```
  </Tab>
</Tabs>

## Verify installation

```bash
sl --version
```

You can also use the full command name:

```bash
securitylayer --version
```

## What's included

The `securitylayer` package installs:

- **`sl` / `securitylayer`** — The CLI binary
- **`@securitylayer/core`** — The security engine (capability gate, taint tracker, normalizer, rules, LLM judge)

## Next steps

Run the interactive setup wizard:

```bash
sl init
```

See [Init](/docs/getting-started/init) for a full walkthrough, or jump to [Quick start](/docs/getting-started/quick-start) for an end-to-end walkthrough.
