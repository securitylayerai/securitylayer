## Systemvoraussetzungen

- **Runtime:** [Bun](https://bun.sh) v1.0 oder neuer
- **OS:** macOS oder Linux
- **Optional:** Docker (für die Datenbank des Web-Dashboards)

## CLI installieren

<Tabs items={["npm", "yarn", "pnpm", "bun"]}>
  <Tab value="npm">
    ```bash
    npm install -g securitylayer
    ```
  </Tab>
  <Tab value="yarn">
    ```bash
    yarn global add securitylayer
    ```
  </Tab>
  <Tab value="pnpm">
    ```bash
    pnpm add -g securitylayer
    ```
  </Tab>
  <Tab value="bun">
    ```bash
    bun add -g securitylayer
    ```
  </Tab>
</Tabs>

## Installation überprüfen

```bash
sl --version
```

Du kannst auch den vollständigen Befehlsnamen verwenden:

```bash
securitylayer --version
```

## Was enthalten ist

Das `securitylayer`-Paket installiert:

- **`sl` / `securitylayer`** — Das CLI-Binary
- **`@securitylayerai/core`** — Die Security-Engine (Capability-Gate, Taint-Tracker, Normalizer, Regeln, LLM-Judge)

## Nächste Schritte

Starte den interaktiven Setup-Wizard:

```bash
sl init
```

Siehe [Init](/de/docs/getting-started/init) für eine vollständige Anleitung oder springe zum [Schnellstart](/de/docs/getting-started/quick-start) für eine Komplettanleitung.
