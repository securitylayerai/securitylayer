## 系统要求

- **运行时：** [Bun](https://bun.sh) v1.0 或更高版本
- **操作系统：** macOS 或 Linux
- **可选：** Docker（用于 Web 仪表盘数据库）

## 安装 CLI

<Tabs items={["npm", "yarn", "pnpm", "bun"]}>
  <Tab value="npm">
    ```bash
    npm install -g securitylayer
    ```
  </Tab>
  <Tab value="yarn">
    ```bash
    yarn global add securitylayer
    ```
  </Tab>
  <Tab value="pnpm">
    ```bash
    pnpm add -g securitylayer
    ```
  </Tab>
  <Tab value="bun">
    ```bash
    bun add -g securitylayer
    ```
  </Tab>
</Tabs>

## 验证安装

```bash
sl --version
```

你也可以使用完整命令名称：

```bash
securitylayer --version
```

## 包含内容

`securitylayer` 包安装以下内容：

- **`sl` / `securitylayer`** — CLI 二进制文件
- **`@securitylayer/core`** — 安全引擎（能力门控、污点追踪器、规范化器、规则、LLM 判断器）
