## Requisitos do sistema

- **Runtime:** [Bun](https://bun.sh) v1.0 ou superior
- **SO:** macOS ou Linux
- **Opcional:** Docker (para o banco de dados do dashboard web)

## Instalar a CLI

<Tabs items={["npm", "yarn", "pnpm", "bun"]}>
  <Tab value="npm">
    ```bash
    npm install -g securitylayer
    ```
  </Tab>
  <Tab value="yarn">
    ```bash
    yarn global add securitylayer
    ```
  </Tab>
  <Tab value="pnpm">
    ```bash
    pnpm add -g securitylayer
    ```
  </Tab>
  <Tab value="bun">
    ```bash
    bun add -g securitylayer
    ```
  </Tab>
</Tabs>

## Verificar a instalacao

```bash
sl --version
```

Voce tambem pode usar o nome completo do comando:

```bash
securitylayer --version
```

## O que esta incluido

O pacote `securitylayer` instala:

- **`sl` / `securitylayer`** — O binario da CLI
- **`@securitylayer/core`** — O motor de seguranca (capability gate, taint tracker, normalizer, rules, LLM judge)

## Proximos passos

Execute o assistente de configuracao interativo:

```bash
sl init
```

Veja [Init](/pt/docs/getting-started/init) para um passo a passo completo, ou va direto para [Inicio rapido](/pt/docs/getting-started/quick-start) para um guia completo em 5 minutos.
